HTML5: Websocket

May 16th, 2012

[The Websocket Protocol](http://www.rfc-editor.org/rfc/rfc6455.txt) The WebSocket Protocol enables two-way communication between a client running untrusted code in a controlled environment to a remote host that has opted-in to communications from that code. The security model used for this is the origin-based security model commonly used by web browsers. The protocol consists of an opening handshake followed by basic message framing, layered over TCP.
